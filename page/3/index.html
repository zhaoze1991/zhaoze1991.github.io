Unique Binary Search Trees I, II

发表于 2015-08-30

Unique Binary Search Trees
递归。helper来计算从begin到end这个区间内，能够组成多少BST。
令root ＝ i， 则它的左子树的区间为begin到i-1, 右子树区间为i+1到end， 一共有左子树个数＊右子树个数个unique BST
注意递归的终止条件。
如果begin ＝＝ end， 说明就一个结点，返回1
如果begin > end， 则表示在这个区间上只有一个null的情况， 返回1.

上述代码超时，因为有很多重复计算。改一改，用一个哈希表来记忆搜索。

也可以用DP来做。
令dp[i]为i个结点构成BST的个数。初始化， dp[0] = 1, dp[1] = 1, 我们来举个栗子。 设i ＝ 3


dp[3] = (dp[0] * dp[2] // 以1为root，左子树结点为0， 右子树结点数为2

+ dp[1] * dp[1] // 以2为root， 左子树结点数为1， 右子树结点数为1
+ dp[2] * dp[0]) // 以3为root，左子树结点数为2， 右子树结点数为0


所以dp[i] = dp[0]*dp[i-1]+dp[1]*dp[i-2]+ …+dp[i-1]*dp[0];
